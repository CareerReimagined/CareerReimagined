Career Reimagined: Milestone Progression

A hybrid career mode that blends mission-gated part unlocks with a redesigned tech tree of optional upgrades. Inspired in part by KSP2's progression approach.

Progression-critical parts are locked behind contract milestones — completing a mission unlocks the exact parts you need for the next one, chosen with care for function and how they build on what came before. The tech tree itself is reserved for optional upgrades you purchase with science from experiments and bonus missions — letting you specialize your campaign. Parts are free, but building capabilities are limited behind Custom Barn kit and building upgrades. Funds exist only for building upgrades. Strategies are disabled for now.

⚠️ Alpha release! This mod is actively in development and playtest feedback is very welcome. Expect rough edges and severe balance tweaks in upcoming versions. Suggest which parts should be unlocked and how! If you hit a bug, please open an issue on GitHub.

Requires: Module Manager, Contract Configurator, Custom Barn Kit

Supports: Making History and Breaking Ground DLCs, ReStock / ReStock+

Drag CareerR folder into GameData folder

On the difficulty selection screen, choose Custom (or modify an existing preset) 3. Set FundsGainMultiplier to 0.1 (10%) 4. Set ScienceGainMultiplier to 0.1 (10%) 5. Other settings can be whatever you prefer
